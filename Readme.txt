#install python
1. Navigate to the below URL:
https://www.python.org/downloads/
Download Python and install it.
NOTE: Select "Add Python to environment variables"

#install required libraries:
1. Once python is successfully installed, open command line in the project folder
as shown in the video.
2. type "pip install flask" and click enter.
3. Flask libraries would be installed.

#Run the project:
1. type "python main.py" and click enter.
2. if it shows that the flask server is running.
3. Copy the HTTP url with port number.
4. Paste that in the browser of your choice and click enter.

Hurray! The app is now running successfully!
